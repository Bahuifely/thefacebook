# thefacebook

Recreación de la versión original de *thefacebook* (2004–2005) utilizando **PHP**, **MySQL**, **HTML**, **CSS** y **JavaScript**.  
Este proyecto tiene fines educativos y busca replicar la apariencia y funcionalidad básica de la plataforma original.

## Características

- Registro de usuarios con validación de correo universitario  
- Inicio de sesión seguro mediante manejo de sesiones  
- Páginas de perfil con información del usuario  
- Página de equipo/creadores  
- Estilo clásico inspirado en la interfaz de thefacebook (2004)

## Requisitos

- PHP **7.4** o superior  
- MySQL **5.7** o superior  
- Servidor web **Apache** o **Nginx**

### Pasos para Desplegar

1. Crear una cuenta en el hosting elegido  
2. Crear una base de datos MySQL  
3. Importar `database-setup.sql`  
4. Subir los archivos del proyecto  
5. Actualizar `config/database.php` con las credenciales  
6. Acceder al dominio proporcionado por el hosting
